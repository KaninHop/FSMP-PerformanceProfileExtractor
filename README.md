# FSMP-PerformanceProfileExtractor
Tool to collect and aggregate performance metrics from FSMP and present it in a useful way


<img width="1927" height="1275" alt="image" src="https://github.com/user-attachments/assets/62d76903-5e6f-46b4-8108-68f03cfd9b6f" />

Features include:

1. Autoloading the logs from your last play session
2. Automatically categorizing the logs imported based on their version
3. Exporting a consolidate image of the charts
4. Exporting as CSV of the raw performance data
